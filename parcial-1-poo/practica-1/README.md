- Descripcion de la clase creada: Se creo la clase "Usuario" con las siguientes caracteristicas:

Atributos privados: nombre y correo
Metodos implementado: Constructor para inicializar los atributos, metodos getters (getNombre(), getCorreo()), metodos setters (setNombre(), setCorreo()).
